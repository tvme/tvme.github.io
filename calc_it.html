Operatori in Python

Torna all'inizio

Nel linguaggio di programmazione Python esistono operatori aritmetici, logici e di confronto.

Addizione:

# Addizione
print(10 + 30)

# Output
>> 40

Sottrazione:

# Sottrazione
print(30 - 20)

# Output
>> 10

Moltiplicazione:

# Moltiplicazione
print(2 * 9)

# Output
>> 18

Divisione:

# Divisione
print(100 / 25)

# Output
>> 4

Oltre ai quattro operatori familiari, ce ne sono alcuni più esotici: modulo (resto della divisione), divisione intera, elevamento a potenza.

Modulo (resto della divisione):

# Modulo della divisione per 2
print(0 % 2)
print(1 % 2)
print(2 % 2)
print(3 % 2)
print(4 % 2)

# Output
>> 0 # 0 % 2
>> 1 # 1 % 2
>> 0 # 2 % 2
>> 1 # 3 % 2
>> 0 # 4 % 2

# Modulo della divisione per 3
print(0 % 3)
print(1 % 3)
print(2 % 3)
print(3 % 3)
print(4 % 3)
print(5 % 3)
print(6 % 3)

# Output
>> 0 # 0 % 3
>> 1 # 1 % 3
>> 2 # 2 % 3
>> 0 # 3 % 3
>> 1 # 4 % 3
>> 2 # 5 % 3
>> 0 # 6 % 3

Divisione intera:

# Divisione intera per 10
print(91 // 10)
print(85 // 10)
print(16 // 10)
print(8 // 10)

# Output
>> 9 # 91 // 10
>> 8 # 85 // 10
>> 1 # 16 // 10
>> 0 # 8 // 10

# Divisione intera per 2
print(14 // 2)
print(15 // 2)
print(7 // 2)
print(6 // 2)

# Output
>> 7 # 14 // 2
>> 7 # 15 // 2
>> 3 # 7 // 2
>> 3 # 6 // 2

Elevamento a potenza:

# Elevamento a potenza del numero 2
print(2 ** 0) # 1
print(2 ** 1) # 2
print(2 ** 2) # 2 * 2
print(2 ** 3) # 2 * 2 * 2
print(2 ** 4) # 2 * 2 * 2 * 2

# Output
>> 1
>> 2
>> 4
>> 8
>> 16

Gli operatori di confronto (╮°-°)╮ possono restituire solo due risultati: True e False.

L'operatore di uguaglianza == restituisce True se i numeri sono uguali, altrimenti False.

a = 10
b = 10
print(a == b)

# Output
>> True

a = 8
b = 7
print(a == b)

# Output
>> False

L'operatore di disuguaglianza != restituisce True se i numeri non sono uguali, altrimenti False.

a = 8
b = 7
print(a != b)

# Output
>> True

L'operatore maggiore > restituisce True se il primo numero è maggiore del secondo, altrimenti False.

a = 8
b = 7
print(a > b)
print(b > a)

# Output
>> True # a > b
>> False # b > a

L'operatore minore < restituisce True se il primo numero è minore del secondo, altrimenti False.

c = 100
d = 200
print(c < d)
print(d < c)

# Output
>> True # c < d
>> False # d < c

L'operatore minore o uguale <= restituisce True se il primo numero è minore o uguale al secondo, altrimenti False.

L'operatore maggiore o uguale >= restituisce True se il primo numero è maggiore o uguale al secondo, altrimenti False.

c = 200
d = 200
print(c >= d)
print(d <= c)

# Output
>> True # c >= d
>> True # d <= c

A volte è necessario eseguire più operatori di confronto contemporaneamente. A tale scopo esiste l'operatore and (operatore di moltiplicazione logica, congiunzione).

print(10 > 0 and 5 > 0)
print(10 % 2 == 0 and 12 % 2 == 0) # entrambi i numeri sono pari

# Output:
>> True
>> True

Se almeno uno degli operatori è False, il risultato dell'operatore sarà False. La congiunzione è vera se e solo se tutte le condizioni sono vere.

print(10 > 100 and 5 > 0 and 10 > 0) # False

# Output:
>> False

L'addizione logica (disgiunzione) o operatore or richiede che ALMENO una condizione sia soddisfatta.

print(10 > 100 or 5 > 100 or 10 > 0) # True
print(1 == 0 or 2 == 0) # False, entrambe le condizioni sono false

# Output:
>> True
>> False

L'ultimo degli operatori è l'operatore di inversione not. L'operatore not cambia (inverte) il valore al suo opposto.

print(not False) # True
print(not True) # False
print(not 2 == 0) # True

# Output:
>> True
>> False
>> True

L'operatore not ha la priorità.

print(not 1 == 0 or 2 == 0) # True, il valore della prima condizione è invertito

# Output:
>> True

Risoluzione dei problemi

1. Stessa parità

Dati due numeri interi: A, B. Verificare la verità dell'affermazione: "I numeri A e B hanno la stessa parità".

# Input:
>> 0
>> 1
# Output:
>> False

# Input:
>> 2
>> 10
# Output:
>> True

2. Uno positivo

Dati tre numeri interi: A, B, C. Verificare la verità dell'affermazione: "Almeno uno dei numeri A, B, C è positivo".

# Input:
>> 0
>> -1
>> -10
# Output:
>> False

# Input:
>> -1
>> 1
>> 0
# Output:
>> True

3. Ultima cifra

Dato un numero naturale. Visualizza la sua ultima cifra.

# Input:
>> 2345678
# Output:
>> 8

# Input:
>> 19
# Output:
>> 9

4. Cifre di un numero a due cifre

Dato un numero a due cifre. Trova la somma delle sue cifre.

# Input:
>> 22
# Output:
>> 4

# Input:
>> 99
# Output:
>> 18

5. Cifre di un numero a tre cifre

Dato un numero a tre cifre. Trova la somma delle sue cifre.

# Input:
>>123
# Output:
>> 6

# Input:
>> 332
# Output:
>> 8

6. Cifre diverse

Dato un numero a tre cifre. Verificare la verità dell'affermazione: "Tutte le cifre di questo numero sono diverse".

# Input:
>> 123
# Output:
>> True

# Input:
>> 332
# Output:
>> False

7. Orologio (boss finale)

Dall'inizio della giornata sono trascorsi N secondi (N - intero). Trova il numero di ore, minuti e secondi sull'orologio elettronico.

# Input:
>> 1000
# Output:
>> 0 16 40

# Input:
>> 10000
# Output:
>> 2 46 40

# Input:
>> 85001
# Output:
>> 23 36 41
