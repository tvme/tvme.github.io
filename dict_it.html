Dizionari in Python

Torna all'inizio

Un dizionario è una struttura dati non ordinata che permette di memorizzare coppie «chiave — valore».

# Creazione di un dizionario tramite letterale
studente = {'nome': 'Marco', 'eta': 12}
credenziali = {'email': 'hacker1337@mail.ru', 'password': '123456'}

# Creazione di un dizionario tramite la funzione dict()
studente = dict(nome='Marco', eta=12)
credenziali = dict(email='hacker1337@mail.ru', password='123456')

Per ottenere il valore di una chiave specifica si usano le parentesi quadre [].

# Otteniamo il valore con la chiave 'nome'
studente = dict(nome='Marco', eta=12)
print(studente['nome']) # -> Marco

L'aggiornamento dei valori esistenti avviene esattamente allo stesso modo.

# Otteniamo il valore con la chiave 'nome'
studente = dict(nome='Marco', eta=12)
studente['nome'] = 'Antonio'

print(studente['nome']) # -> Antonio

Per eliminare una chiave e il valore corrispondente da un dizionario si può usare del

# Eliminazione della chiave 'eta'
studente = dict(nome='Marco', eta=12)
del studente['eta']

print(studente) # -> {'nome': 'Marco'}

Il metodo get() restituisce il valore per la chiave specificata. Se la chiave specificata non esiste, il metodo restituirà None. È anche possibile specificare un valore predefinito che verrà restituito al posto di None se la chiave non si trova nel dizionario.

# Utilizzo del metodo get()
studente = dict(nome='Marco', eta=12)

print(studente.get('nome'))     # -> Marco
print(studente.get('cognome')) # -> None
print(studente.get('cognome', 'Nessuna chiave')) # -> Nessuna chiave

Il metodo pop() elimina la chiave e restituisce il valore corrispondente.

# Utilizzo del metodo pop()
sneakers = dict(marca='Adidas', prezzo='110 €', modello='Nite Jogger')

modello = sneakers.pop('modello')

print(sneakers) # -> {'marca': 'Adidas', 'prezzo': '110 €'}
print(modello)    # -> Nite Jogger

Il metodo keys() restituisce una speciale collezione di chiavi nel dizionario.

# Utilizzo del metodo keys()
sneakers = dict(marca='Adidas', prezzo='110 €', modello='Nite Jogger')

print(sneakers.keys()) # -> dict_keys(['marca', 'prezzo', 'modello'])
# dict_keys - è una collezione immutabile di elementi.

chiavi = list(sneakers.keys())
print(chiavi) # -> ['marca', 'prezzo', 'modello']

Il metodo values() restituisce una speciale collezione di valori nel dizionario.

# Utilizzo del metodo values()
sneakers = dict(marca='Adidas', prezzo='110 €', modello='Nite Jogger')

print(sneakers.values()) # -> dict_values(['Adidas', '110 €', 'Nite Jogger'])
# dict_values - è una collezione immutabile di elementi.

valori = list(sneakers.values())
print(valori) # -> ['Adidas', '110 €', 'Nite Jogger']

Il metodo items() restituisce coppie «chiave — valore» in formato di tuple.

# Utilizzo del metodo items()
sneakers = dict(marca='Adidas', prezzo='110 €', modello='Nite Jogger')

print(sneakers.items()) # -> dict_items([('marca', 'Adidas'), ('prezzo', '110 €'), ('modello', 'Nite Jogger')])
# dict_items - è una collezione immutabile di elementi.

items = list(sneakers.items())
print(items) # -> [('marca', 'Adidas'), ('prezzo', '110 €'), ('modello', 'Nite Jogger')]

sneakers = dict(marca='Adidas', prezzo='110 €', modello='Nite Jogger')

# Stampa delle chiavi del dizionario con un ciclo for
for chiave in sneakers:
    print(chiave)

# -> marca
# -> prezzo
# -> modello
# Stampa dei valori del dizionario con un ciclo for
for valore in sneakers.values():
    print(valore)

# -> Adidas
# -> 110 €
# -> Nite Jogger
# Stampa di chiavi e valori del dizionario con un ciclo for
for chiave, valore in sneakers.items():
    print(chiave, valore)

# -> marca Adidas
# -> prezzo 110 €
# -> modello Nite Jogger

Il metodo setdefault() restituisce il valore della chiave, ma se non esiste, crea la chiave con il valore specificato (predefinito None).

# Metodo setdefault()

studente = dict(nome='Marco', eta=12)
studente.setdefault('cognome', 'Rossi')

print(studente) # -> {'nome': 'Marco', 'eta': 12, 'cognome': 'Rossi'}

Risoluzione dei problemi

1. Utente

# L'utente inserisce nome, cognome, età. Crea un dizionario utente e registra i dati dell'utente in esso.
nome = input('Inserisci il tuo nome: ')
cognome = input('Inserisci il tuo cognome: ')
eta = input('Inserisci la tua età: ')

utente = dict(nome=nome, cognome=cognome, eta=eta)
print(utente)

2. Trova parola

# Stampa la parola più frequente nella stringa inserita.
lista_di_parole = ['ciao', 'ciao', 'salve']
parole = {}

for parola in lista_di_parole:
    parole[parola] = parole.get(parola, 0) + 1

# La funzione max può ricevere una funzione come secondo argomento
parola_freq = max(parole, key=parole.get)

print(parola_freq)

Problemi da risolvere autonomamente

1. Frutta

L'utente inserisce un numero K - la quantità di frutta. Quindi inserisce K frutti nel formato: nome del frutto e sua quantità. Aggiungi tutti i frutti al dizionario, dove il nome del frutto è la chiave e la quantità è il valore. Per esempio:

# Input:
>> 3 # Quantità di frutta
>> Mela
>> 3
>> Arancia
>> 3
>> Mandarino
>> 10
# Output:
>> {'Mela': 3, 'Arancia': 3, 'Mandarino': 10}

2. Maggiore e minore

L'utente inserisce un numero N. Quindi inserisce i dati personali (nome ed età) dei suoi N amici. Crea una lista friends e aggiungi ad essa N dizionari con le chiavi name ed age. Trova il più giovane degli amici e stampa il suo nome.

# Input:
>> 3 # Numero di amici
>> Marco
>> 11
>> Alessandro
>> 12
>> Lorenzo
>> 10
# Output:
>> Lorenzo

3. Ancora un po' sugli amici

L'utente inserisce un numero N. Quindi inserisce i dati personali (nome ed età) dei suoi N amici. Crea una lista amici e aggiungi ad essa N dizionari con le chiavi nome ed eta. Stampa l'età media di tutti gli amici e il nome più lungo.

# Input:
>> 3 # Numero di amici
>> Marco
>> 12
>> Gabriele
>> 20
>> Lorenzo
>> 10
# Output:
>> 14
>> Gabriele

4. Dizionario inglese

"È ora di imparare l'inglese", si disse Stepa e decise di scrivere un programma per imparare l'inglese. Il programma riceve in input una parola in inglese e diverse sue traduzioni in russo. Componi un dizionario in cui la chiave è la parola inglese e il valore è una lista di parole russe. In questo problema è necessario utilizzare il metodo split() delle stringhe.

# Input:
>> 4 # Numero di parole
>> apple - mela
>> orange - arancione, arancia
>> grape - uva, acino
>> easy - semplice, facile, non difficile, comodo, non complicato
# Output:
>> {'apple': ['mela'], 'orange': ['arancione', 'arancia'], 'grape': ['uva', 'acino'], 'easy': ['semplice', 'facile', 'non difficile', 'comodo', 'non complicato']}
