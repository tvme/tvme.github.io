Operatore condizionale in Python

Torna all'inizio

L'operatore condizionale esegue una determinata azione se la condizione specificata è vera (uguale a True).

# Presta attenzione all'indentazione di quattro (4!) spazi
# o una tabulazione (tasto TAB)
if True: # Due punti obbligatori prima del blocco di codice
    print('La condizione è vera, quindi verrò stampato')

# Output
>> La condizione è vera, quindi verrò stampato

Nella condizione, qualsiasi tipo di dati viene implicitamente convertito in bool. Una stringa vuota e zero sono uguali a False.

print(bool(1)) # -> True
print(bool(0)) # -> False
print(bool(10.2)) # -> True
print(bool('')) # -> False
print(bool('Hello')) # -> True

Se la condizione è falsa, il codice sotto l'operatore if non verrà eseguito.

if 0: # Due punti obbligatori prima del blocco di codice
    print('Non verrò mai stampato :(')

Se la condizione in if è falsa, verrà eseguito l'operatore sotto l'operatore else.

if 0:
    print('Non verrò mai stampato :(')
else:
    print("L'operatore else entra in gioco")
    # Per utilizzare un apostrofo in apice, è necessario racchiudere l'intera riga in un doppio apostrofo.

# Output
>> L'operatore else entra in gioco

A volte due opzioni non sono sufficienti, per questo esiste l'operatore elif, e per tutto il resto c'è...

age = int(input())
if age < 10:
    print('La tua età è una cifra singola :)')
elif age < 15:
    print('In bocca al lupo con la geometria :):):)')
else:
    print('Sei già abbastanza grande :(')

Se la condizione nell'operatore if è falsa, l'interprete controlla la verità della condizione degli operatori elif, tutti in ordine. Se tutte le condizioni sono false, verrà eseguito l'operatore else.

Esempi

1. Pari-dispari

# Stampa la stringa 'pari' se il numero inserito è pari, e la stringa 'dispari' se il numero è dispari.
a = int(input())
if a % 2 == 0:
    print('pari')
else:
    print('dispari')

# Input
>> 10
# Output
>> pari

# Input
>> 11
# Output
>> dispari

2. Molti elif

# Se nel tuo codice ci sono più di tre elif, significa che qualcosa non va e dovresti pensare a un nuovo algoritmo.
# Questo è un esempio puramente accademico, non dovresti scrivere così.
a = int(input('Inserisci un numero da 1 a 100'))
if a < 10:
    print('Il tuo numero è minore di 10')
elif a < 20:
    print('Il tuo numero è minore di venti')
elif a < 30:
    print('30 è il limite massimo')
elif a < 40:
    print('Il tuo numero è minore di 40')
elif a < 50:
    print('Molto, ma non più di cinquanta')
elif a < 60:
    print('Il numero è minore di sei*dieci')
elif a < 70:
    print('Il tuo numero si trova nell'ottava decina')
elif a < 80:
    print('Il tuo numero è minore di 80')
elif a < 90:
    print('Il tuo numero si trova nella nona decina')
elif a < 100:
    print('Cento è maggiore del tuo numero')
else:
    print('Astuto, ma non mi inganni, il numero è troppo grande')

# Input
>> 11
# Output
>> Il tuo numero è minore di venti

# Input
>> 91
# Output
>> Cento è maggiore del tuo numero

# Input
>> 58
# Output
>> Il numero è minore di sei*dieci

3. Bianca o nera

# L'utente inserisce due numeri: le coordinate di una casa della scacchiera.
# Stampa YES se la casa è bianca, e NO se è nera.
x = int(input('Inserisci la coordinata x: '))
y = int(input('Inserisci la coordinata y: '))
# L'idea è questa: se la parità delle coordinate coincide,
# allora è una casa nera, altrimenti è bianca.
if (x + y) % 2 == 1:
    print('YES')
else:
    print('NO')

# Input:
>> 1
>> 1
# Output:
>> NO

# Input:
>> 1
>> 1
# Output:
>> NO

4. Bianca o nera (coordinate casuali)

# Stesso problema, ma con numeri casuali.
# Le coordinate della casa della scacchiera vengono scelte casualmente
# Stampa YES se la casa è bianca, e NO se è nera.
from random import randint # importazione della funzione randint
x = randint(1,8) # la funzione randint restituirà un numero casuale da 1 a 8
y = randint(1,8)
print(x, y) # stampa della coppia di numeri casuali
if (x + y) % 2 == 1:
    print('YES')
else:
    print('NO')

Risoluzione dei problemi

1. Minore di due

L'utente inserisce due numeri interi. Stampa il minore dei due.

2. A quattro cifre?

L'utente inserisce un numero intero. Stampa YES se questo numero è a quattro cifre, altrimenti NO.

3. Triangolo?

Un triangolo esiste solo quando la somma di due lati qualsiasi è maggiore del terzo. Dati a, b, c - lati del presunto triangolo. È necessario confrontare la lunghezza di ciascun segmento-lato con la somma degli altri due. Se almeno in un caso un segmento risulta maggiore della somma degli altri due, allora un triangolo con tali lati non esiste.

# Input:
>> 3
>> 4
>> 5
# Output:
>> YES

# Input:
>> 3
>> 10
>> 5
# Output:
>> NO

4. Momento della giornata

L'utente inserisce l'ora in ore. Se il numero inserito è compreso tra 5 e 11 ore inclusi, stampa la stringa 'Mattina'. Se il numero è compreso tra 12 e 17 ore inclusi, stampa la stringa 'Giorno'. Se il numero è compreso tra 18 e 22, stampa 'Sera'. In caso di notte fonda (da 23 a 4 inclusi) stampa 'Notte'. Negli altri casi stampa la stringa 'Errore'.

# Input:
>> 3
# Output:
>> Notte

# Input:
>> 6
# Output:
>> Mattina

# Input:
>> 13
# Output:
>> Giorno

# Input:
>> 2020
# Output:
>> Errore

5. Giorno della settimana

L'utente inserisce il numero del giorno della settimana. Stampa la stringa 'Fine settimana' se il numero inserito è 6 o 7. Se il numero è compreso tra 1 e 5 inclusi, stampa la stringa 'Giorni feriali'.

# Input:
>> 3
# Output:
>> Giorni feriali

# Input:
>> 6
# Output:
>> Fine settimana

6. Descrizione del numero

L'utente inserisce un numero intero. Stampa la sua descrizione sotto forma di stringa come "numero pari negativo", "zero", "numero dispari positivo", ad esempio, la descrizione numerica del numero 190 è la stringa "numero pari positivo".

# Input:
>> 1
# Output:
>> numero dispari positivo

# Input:
>> -6
# Output:
>> numero pari negativo

# Input:
>> 0
# Output:
>> zero

# Input:
>> 2
# Output:
>> numero pari positivo

7. Cavallo (boss finale)

Il cavallo degli scacchi si muove a forma di "L" - due case in verticale in qualsiasi direzione e una casa in orizzontale, o viceversa. Date due diverse case della scacchiera, determina se il cavallo può passare dalla prima casa alla seconda in una mossa. Se almeno uno dei numeri inseriti non è compreso tra 1 e 8, stampa la stringa "Errore!".

# Input:
>> 1 # coordinata x1
>> 1 # coordinata y1
>> 2 # coordinata x2
>> 2 # coordinata y2
# Output:
>> NO

# Input:
>> 1 # coordinata x1
>> 1 # coordinata y1
>> 3 # coordinata x2
>> 2 # coordinata y2
# Output:
>> YES

# Input:
>> 3 # coordinata x1
>> 8 # coordinata y1
>> 2 # coordinata x2
>> 6 # coordinata y2
# Output:
>> YES

# Input:
>> 3 # coordinata x1
>> 4 # coordinata y1
>> 5 # coordinata x2
>> 6 # coordinata y2
# Output:
>> YES
