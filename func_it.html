Funzioni in Python

Torna all'inizio

Una funzione è un blocco di codice nominato a cui è possibile fare riferimento da qualsiasi punto del programma. Una funzione ha un nome e un elenco di parametri di input, nonché un valore restituito.

Una funzione consente di utilizzare lo stesso frammento di codice più volte in un programma.

La dichiarazione di una funzione in Python appare così:

def nome_funzione(argomento1, argomento2, ...):
    # codice della funzione
# def - DEclare Function - "dichiarare una funzione"
# nome_funzione - nome della funzione
# (argomento1, argomento2, ...) - elenco di argomenti passati alla funzione al momento della sua chiamata
# corpo della funzione - è tutto il codice che segue i due punti

# Dichiarazione della funzione
def saluto(nome):
    print('Ciao, ' + nome)

# Chiamate alla funzione

saluto('Max')
saluto('Marco')
saluto('Alex')
saluto('Luciana')

# Output
>> Ciao, Max
>> Ciao, Marco
>> Ciao, Alex
>> Ciao, Luciana

L'operatore return restituisce un valore dalla funzione.

Immagina una normale calcolatrice. Inserisci il primo numero, l'operazione e il secondo numero. La calcolatrice ci restituisce il risultato dell'operazione su questi numeri. Semplice, vero? Le funzioni allo stesso modo sanno restituire un valore usando l'operatore speciale return.

# Dichiarazione della funzione
def somma2(a, b):
    return a + b

# Chiamate alla funzione
s1 = somma2(10, 2)
s2 = somma2(108, 100)
s3 = somma2(3, 1)

print(f's1 = {s1}')
print(f's2 = {s2}')
print(f's3 = {s3}')

>> s1 = 12
>> s2 = 208
>> s3 = 4

# Funzione di moltiplicazione di due numeri
def molt2(a, b):
    return a * b

# Chiamiamo la nostra funzione
m1 = molt2(10, 2)
m2 = molt2(108, 100)
m3 = molt2(3, 1)

print(f'm1 = {m1}')
print(f'm2 = {m2}')
print(f'm3 = {m3}')

>> m1 = 20
>> m2 = 10800
>> m3 = 3

Per i parametri della funzione è possibile specificare valori predefiniti. Ciò consente di chiamare la funzione con un numero inferiore di parametri.

# L'argomento nome per impostazione predefinita è 'mondo'
def saluto(nome='mondo'):
    print('Ciao, ' + nome)

saluto()
saluto('Marco')

>> Ciao, mondo
>> Ciao, Marco

4. Somma della lista

# Scrivere una funzione amountList che restituisce la somma di tutti gli elementi della lista.
def amountList(lst):
    amount = 0
    for x in lst:
        amount += x
    return amount

print(amountList([1, 2, 3]))

mylist = [1, 2, 4, 8, 16]
s = amountList(mylist)

print(f'La somma della lista {mylist} è uguale a {s}')

>> 6
>> La somma della lista [1, 2, 4, 8, 16] è uguale a 31

5. Fibonacci

# Scrivere una funzione fib che restituisce l'n-esimo numero di Fibonacci.
# La sequenza di Fibonacci appare così: 1 1 2 3 5 8 13 21 34
def fib(n):
    a, b = 0, 1
    if n == 0: return 0
    for i in range(1, n):
        a, b = b, a + b
    return b

print(fib(2))
print(fib(3))
print(fib(4))
print(fib(5))
print(fib(10))

>> 1
>> 2
>> 3
>> 5
>> 55

6. Fattoriale

# Scrivere una funzione fact che calcola il valore del fattoriale del numero N.
# Il fattoriale di un numero è il prodotto di tutti i numeri da 1 a N.
# Ad esempio, il fattoriale del numero 5 è uguale a 120 (5! = 120).
def fact(n):
    result = 1
    while n > 1:
        result *= n
        n -= 1
    return result

print(fact(2))
print(fact(3))
print(fact(4))
print(fact(5))

>> 2
>> 6
>> 24
>> 120
