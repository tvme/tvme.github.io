Input e output dei dati

Torna all'inizio

Una variabile è un contenitore nominato per un valore specificato.

# Esempio di creazione di variabili
age = 19
name = 'Lorenzo'
isActive = True

name = 'Marco'
Name = 'Lorenzo'

# name e Name sono variabili diverse

# I nomi delle variabili non possono iniziare con un numero!

In Python ci sono 4 tipi di dati primitivi:

# int (numeri interi)
age = 18

# float (numeri decimali)
fraction = 2.5

# str (stringhe)
fruit = 'apple'

# bool (vero o falso)
isReady = True # solo due valori: True e False

Possiamo convertire un tipo di dati in un altro usando funzioni con lo stesso nome. Ad esempio, un numero può diventare una stringa, una stringa un numero, un numero decimale un intero.

age = '22' # str -> '22'
age = int(age) # int -> 22
age = float(age) # float -> 22.0
age = bool(age) # bool -> True

La funzione print() visualizza i dati sullo schermo.

name = 'Alessandro'
# Tra parentesi vengono scritti i parametri passati alla funzione
print(name)

# Output
>> Alessandro

La funzione print() può accettare più parametri di input.

print(1, 2, 3)

# Output
>> 1 2 3

Ogni print() visualizza i dati su una nuova riga. Per impostazione predefinita, il carattere di terminazione della stringa è uguale al carattere di nuova riga (\n).

print('Hello')
print('world')

# Output
>> Hello
>> world

Il carattere di terminazione della stringa nella funzione print() può essere modificato.

print('Hello', end=' ')
print('world')

# Output
>> Hello world

La funzione input() accetta l'input di dati da parte dell'utente.

name = input()
print('Hello ' + name)
# Dopo l'avvio dello script, Python attenderà l'input dei dati

La funzione input può accettare solo un argomento: una stringa che verrà visualizzata prima della stringa di input.

name = input('Enter your name: ')
print('Hello, ' + name)

I dati ottenuti con la funzione input() sono di tipo stringa (str).

Le stringhe possono essere sommate tra loro, tale somma è chiamata concatenazione o unione.

# Somma di due numeri stringa
number1 = input('Inserisci un numero: ')
number2 = input('Inserisci un numero: ')
print(number1 + number2)

# Input:
>> 1
>> 2
# Output:
>> 12

Convertiamo il tipo stringa in un numero intero (str -> int).

# Somma corretta di due numeri
number1 = int(input('Inserisci un numero: '))
number2 = int(input('Inserisci un numero: '))
print(number1 + number2)

# Input:
>> 1
>> 2
# Output:
>> 3

Esempi

1. Prodotto

# Prodotto di due numeri inseriti
a = int(input('Inserisci un numero: '))
b = int(input('Inserisci un numero: '))
print(a * b)

# Input:
>> 4
>> 3
# Output:
>> 12

2. Saluto

# Saluto all'utente con il suo nome
firstname = input('Inserisci il tuo nome: ') # qui non è necessario convertire in int
lastname = input('Inserisci il tuo cognome: ')
print('Buongiorno, ' + firstname + ' ' + lastname) # non dimenticare lo spazio tra le parole

# Input:
>> Lorenzo
>> Rossi
# Output:
>> Buongiorno, Lorenzo Rossi

3. Resto

# L'operazione % permette di ottenere il resto della divisione
print(10 % 2) # 0, poiché 10 è divisibile per 2 senza resto
print(10 % 3) # 1, il resto è 1
print(10 % 4) # 2, il resto è 2

# Output:
>> 0
>> 1
>> 2

4. Divisione intera

# L'operazione // permette di ottenere la parte intera della divisione
print(10 // 2) # 5
print(10 // 3) # 3
print(10 // 4) # 2

# Output:
>> 5
>> 3
>> 2
